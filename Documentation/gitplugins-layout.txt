gitplugins-layout(5)
===============

NAME
----
gitplugins-layout - describes organisation of directories used by package managers and plugin authors to install plugins


DESCRIPTION
-----------
This document describes where to place artifacts of a plugin in order that the git runtime may find them.

It will cover topics such as where to place scripts and shell libraries, where to place man pages and where
to place HTML documents.
