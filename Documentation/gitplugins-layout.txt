gitplugins-layout(5)
===============

NAME
----
gitplugins-layout - describes organisation of directories used for plugin management.


DESCRIPTION
-----------
This document will describe the structure of the plugins/ directory amd the structure of the
active/ directory. The active/ directory has a bin/ directory in the PATH and a man/ directory in the
MANPATH. These directories contain symbolic links back to files in subdirectories of the plugins/ directory.

The symbolic links are managed by the activate and deactivate subcommands of the git-plugin command.
